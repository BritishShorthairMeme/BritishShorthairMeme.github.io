Toilet Protocol

Toilet Protocol is a cryptocurrency that aims to recycle the trash tokens that exist on the Polygon Mainnet network. Too many tokens that are no longer useful on the Blockchain such as fraudulent tokens, rugpull, tokens that cannot be traded on the market and so on.

This protocol aims to help the community or wallet owners have a happy feeling without having to mourn useless trash tokens.

The mechanism of this protocol is that the trash token that is owned is sent to the Toilet Protocol smart contract so that this smart contract will print a new token called TOI

How is the TOI token different from other tokens?

TOI tokens can only be issued from the trash tokens they send to the Toilet Protocol smart contract, so that this TOI token can be owned by anyone without exception.
