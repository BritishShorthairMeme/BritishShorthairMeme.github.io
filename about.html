Toilet Protocol

Toilet Protocol is a cryptocurrency that aims to recycle the trash tokens that exist on the Polygon Mainnet network. Too many tokens that are no longer useful on the Blockchain such as fraudulent tokens, rugpull, tokens that cannot be traded on the market and so on.

This protocol aims to help the community or wallet owners have a happy feeling without having to mourn useless trash tokens.

The mechanism of this protocol is that the trash token that is owned is sent to the Toilet Protocol smart contract so that this smart contract will print a new token called TOI

How is the TOI token different from other tokens?

TOI tokens can only be issued from the trash tokens they send to the Toilet Protocol smart contract, so that this TOI token can be owned by anyone without exception.

Are these TOI tokens valuable?

It all depends on the community that owns the TOI token, whether they want to add to the market pool or not, because the developer will not add the liquidity pool in the DEX market. Because developers don't want to be central to controlling TOI tokens, these TOI tokens are community based, so the community decides. Protocol Toilet Developer is just a bridge.

What is the total supply of TOI tokens?

TOI tokens will only be issued as much as 990,000,000,000 TOI

What happens after the TOI tokens finish minting up to 990,000,000,000?

This will be a benefit for TOI token holders, because every shipping transaction will get the transaction fee.

Why Protocol Toilets use the Polygon Mainnet network?

Because the transaction fees carried out do not require very expensive fees and also have speed in transacting on the Blockchain. Unlike Bitcoin, Ethereum and Binance coins, the Polygon Mainnet network is an alternative for transactions, thus helping people in their daily activities.
